# **Module 7 Assignment 1**
The code here AverageCalculator.java contains at least four deliberate flaws (malfunctions). Goals of this repository is to conduct debugging, defensive programming and logging.

## **Prerequisites**
- Made using Java Version 24.0.2.
- Java JDK 17+ needed.

## **Code Features**
- Takes the average of the numbers that are found within a hardcoded list of inputs
- Logging each parsed number that goes through the method

## **Design & Lecture Principles Implemented**
- Scientific Debugging
- Defensive Programming & Logging

## **Testing Suite**
Currently, this project does not utilize a separate unit test framework. To test this branch, one can utilize the file AverageCalculator.java as there is a Main method within there.

## **How To Run**
**Via Java IDE:**
1) Download repository or pull branch
2) Open your project folder holding the file
3) Run the AverageCalculator.java file
